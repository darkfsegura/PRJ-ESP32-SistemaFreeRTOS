void setup ( )

Función de configuración inicial del sistema Arduino.

  • Inicializa comunicación serial
  • Crea semáforos y mutex para sincronización
  • Crea cola de mensajes para comunicación inter-tareas
  • Crea y lanza todas las tareas FreeRTOS

Esta función se ejecuta una vez al inicio del programa
